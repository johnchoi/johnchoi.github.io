1. 产品概述

1.1 名词说明

名称 说明
Match Makin预约系统 Match Making预约系统是为了促进展会中展商和买家能够尽可能多地成功达成商业协议而生的匹配系统
完美配对 展商和买家互相选择了对方的情况
买家优先 买家向展商发出了邀约的情况
展商优先 展商向买家发出了邀约的情况

1.2 产品概述及目的

Match Making展会商业会谈预约系统是为了促进展会中展商和买家能够尽可能多地成功达成商业协议而生的匹配系统，会谈预约系统能够在展前3-6个月内就将所有的展商与买家进行匹配搓合，并约定洽谈时间，在展会现场，买家可以根据自己的约谈日程表前往展商的展台进行商业洽谈，减少了买卖双方在展会中的时间浪费，使约谈更有效率，时间更紧凑。

Match Making展会商业会谈预约系统将预约分为三个不同的阶段：1. PSA(Pre-Scheduling Appointment)展前预约阶段；2. OD(Online Diary)在线互动阶段；3. BMS现场商务约谈安排阶段。

在PSA阶段，展商和买家互相选择一批想要约见的对象，PSA阶段截止后，系统会按照设定的顺序（完美配对 => 买家优先 => 展商优先，买家和展商的顺序可在后台调整）自动匹配商务约谈时间。

在OD阶段，展商和买家可以通过在线互动补全剩余的商务约谈空档。在线互动平台将给买家和展商在管理商务约谈日程安排上更大的自由度，展商和买家可以在在线互动平台系统上看到来自对方的预约邀请，并通过选择一个共同的空档向对方发出商务约谈邀请。

在BMS阶段，展商和买家可以通过现场的买家-约见-卖家（BMS）系统来针对现场出现的意外情况对约谈行程做最后的补充和调整。

2.2 功能详情

2.2.1 展会预约系统设置

2.2.1.1 开启展会预约系统
  • 简要说明

开启展会预约系统功能用于初始化预约系统所需要的数据，包括参会人的展商和买家角色以及角色对应的规则，展商及买家通道。

  • 业务规则

  1. 在未初始化预约系统时，会议中不存在展商/买家角色以及角色规则
  2. 在未初始化预约系统时，预约系统中的其他设置都无法使用（禁用）
  3. 选择使用通道定义展商/买家角色
  4. 第二步有两个选项：“使用通道定义展商/买家角色规则”，“自定义角色规则”，默认为“使用通道定义展商/买家规则”
  5. 选择“使用通道定义展商/买家角色规则”时，系统将自动为用户创建通道、角色以及设置角色规则，创建完成后将跳回设置页面
  6. 选择“自定义角色规则”时，系统将自动为用户创建角色，并跳转到角色设置页面。

  7. 角色设置页面交互：从预约设置页面进入角色设置页面时，需要带上参数，角色设置页面判断参数并引导设置展商/买家角色规则，设置完成后引导返回“预约系统设置”

  • 在未初始化预约系统时，预约系统中的其他设置都无法使用（禁用）
    ￼

  • 角色设置页面引导
    ￼

2.2.1.2 增加两种固定的角色
  • 简要说明

单会参会人角色中增加两种固定的角色类型：展商和买家，这两种角色类型仅用于实现预约系统中的特殊业务逻辑。

  • 业务规则

  1. 在未初始化预约系统时，单会系统中不存在这两种角色，且这两种角色无法手动创建，必须通过初始化预约系统进行创建
  2. 这两种角色可以在创建成功后修改其名称，但并不影响实际使用，由于预约系统中将会使用这两个角色名称，所以修改名称会影响前台显示
2.2.1.3 预约系统基础设置
  • 简要说明

基础设置主要包括是否开启预约系统，以及设置三个不同阶段的起止时间

  • 业务规则

  1. 开启/关闭预约系统仅控制展商和买家在个人中心中是否能够看到预约系统的菜单，不涉及任何预约数据的变动
  2. 预约系统初始化完成后，默认处于开启状态
  3. 三个阶段的起止时间只需要精确到分钟
  4. PSA阶段开始时间没有任何限制，可以选择任何时间；结束时间必须晚于开始时间，且与OD阶段的开始时间必须间隔2小时以上，即结束时间必于早于OD阶段开始时间减去2小时，当OD阶段开始时间为空时，PSA阶段则没有任何限制。
  5. OD阶段开始时间必于晚于PSA阶段结束时间2小时后，OD阶段结束时间必须早于或等于BMS开始时间。
  6. BMS阶段开始时间默认为会议开始时间，结束时间默认为会议结束时间，BMS开始时间不能早于OD结束时间。
  7. 修改BMS阶段的开始和结束时间时，如果“场次/时间”尚未设置，则需要同步更新“场次/时间”设置中的开始和结束日期。
  8. 预约取消时间：默认为5天，预约请求提交后，在该时间段后如果没有被对方同意，预约将自动取消

2.2.1.5 设置场次/时间
  • 简要说明

场次/时间设置主要设置现场约谈的起止日期，每天的开始和结束时间以及每场约谈的时长，并能过这些设置计算出每天的约谈场次时间段。

  • 业务规则

  1. 现场约谈默认开始时间为会议开始时间，默认结束日期为会议结束时间。
  2. 修改BMS阶段的开始和结束时间时，如果“场次/时间”尚未设置，则需要同步更新“场次/时间”设置中的开始和结束日期。
  3. 修改场次/时间中的日期，不会反向影响基础设置中BMS阶段的开始和结束时间。
  4. “场次/时间”中的开始日期必须晚于或等于OD阶段的结束日期。
  5. 现场约谈日期确定后，下方自动按日期生成每天上午和下午的开关，用户可以自己选择是否开启或关闭某个时间段。
  6. 默认上午开始时间为09:00，上午结束时间为12:00；下午开始时间为13:00，下午结束时间为18:00。
  7. 上午的开始时间和结束时间不得晚于12:00，且上午结束时间不得早于上午的开始时间。
  8. 下午开始时间不得早于上午的结束时间，下午结束时间不得早于下午的开始时间，下午结束时间不得晚于每天的00:00
  9. “启用午餐时间”：默认开启，启用午餐时间后，会将上午结束时间到下午开始时间之间的时间也设置为一个时间段，可针对该时间段开放预约。
  10. 启用午餐时间时，如果某一天的上午或下午被禁用，则不生成该天的午餐时间段。
  11. 每场约谈时长默认为20分钟，限制为大于0的正整数。
  12. 点击“开始设置”，按上述设置生成约谈时间段。
2.2.1.6 设置PSA阶段匹配优先规则
  • 简要说明

使用该设置可以确定在PSA阶段结束后，系统自动匹配的规则，以及最多可以取消的预约数

  • 业务规则

  1. 完美匹配永远排在第一位，无法更改其匹配顺序
  2. 如果选择了买家优先，则在完美匹配后优先匹配买家选择了展商的预约数据
  3. 如果选择了展商优先，则在完美匹配后优先匹配展商选择了买家的预约数据

  4. 最多取消预约数：默认为3个，控制在在线互动阶段可以取消的预约数量，仅能取消PSA阶段匹配成功的预约，无法取消在线互动阶段手动匹配成功的预约

  • 匹配算法
2.2.1.7 互动设置

控制在线互动阶段，双方是否能够互相发送消息，以及展商和买家互相能够查看哪些信息。

  5. 可使用公司名称、联系人姓名和标题进行搜索
  6. 点击某条消息标题或操作中的“查看”按钮时，进入查看消息详情页面
  7. 删除某条消息时，需要二次确认，文案：您确定要删除该消息吗？删除后将无法恢复
  8. 列表页默认显示条数为10条/页，可选范围是10/25/50/100
  9. 进入消息详情页可以对消息进行回复，回复时跳转到发送消息页面，且必需将原消息内容作为引用内容
  10. 发件箱中显示发出的邮件列表，其他功能与收件箱相同
  11. 发送消息页面中，可以选择向多个对象发送消息，在输入框中进行输入时需要在近期联系人中进行搜索，显示自动完成框
  12. 点击添加收件人是，可以在展商或者买家中选择收件人，同时勾选展商和买家时，则是发送给所有人。
  13. 收件人中需要有两个组：展商和买家，当所有展商都被选中时，就显示“展商”，买家也一样
  14. 在添加收件人的窗口中，可以使用搜索功能搜索展商或买家，搜索字段：姓名和公司名称
  15. 发消息时的“系统消息”默认勾选，勾选了系统消息后的发出的消息，将带有“系统消息”的标签

  • 业务规则

  1. 点击“编辑”时，跳到单会的编辑信息页面
2.2.1.12 展前预约：选择预约对象
  • 简要说明

在报名截止后，系统将进入展前预约阶段（PSA），在该阶段用户无法看到“我的预约”以及“进行预约”菜单，展前预约阶段展商和买家可以选择一批预约对象，在展前预约阶段结束后，系统将会按照选择结果进行自动匹配。

  • 业务规则

  1. 需选择预约对象数量设置为与后台生成的“场次/时间”时间段数量一样，如后台生成24个场次，则需要选择的预约对象数量则为24个
  2. 点击选择预约对象后，进入搜索页面，可以按照公司或联系人关键字和所在国家搜索预约对象，展商仅能搜出买家，买家仅能搜出展商
  3. 国家为多选下拉菜单，可以选择多个国家
  4. 点击“重置”会重置搜索条件，并刷新下方列表
  5. 在预约对象列表中可以查看预约对象的信息
  6. 点击预约则将预约对象放置到右侧预约名单中，不实时保存，点击提交时才会保存
2.2.1.13 展前预约：系统根据选择结果自动匹配
  • 简要说明

展前预约阶段结束后，系统将对展商和买家的选择结果进行自动匹配，匹配规则顺序：完美配对（永远优先） => 后台设置（买家优先或展商优先）

  • 业务规则
2.2.1.14 查看对方公司信息
  • 简要说明

在展商或买家搜索到预约对象后，可以查看预约对象的报名信息（包含公司信息）

  • 业务规则

  1. 可查看的报名信息可以在后台“互动设置”中进行设置
2.2.1.15 我的预约：预约日程
  • 简要说明

可看到当前我的整个预约日程表，以及预约完成进度，并可以在这里取消PSA阶段自动匹配成功的预约。取消数量取决于后台设置

  • 业务规则

  1. 预约日程进度：已预约时间段数量/总预约时间段
  2. 取消预约日程取决于后台PSA阶段设置中设置的可取消数量，默认3次
  3. 操作：发送消息，将当前的公司信息带到发送信息页面作为收件人
  4. 操作：取消预约，仅能取消PSA阶段自动匹配成功的预约，在线互动阶段手动匹配的预约无法取消（禁用）
  5. 如果取消预约数量已达上线，则禁用所有取消按钮
  6. 空床的时间段需要标记为空闲
2.2.1.16 我的预约：收到的预约请求
  • 简要说明

查看当前用户自己收到的预约请求，并可查看对方信息，发送消息以及接受和拒绝

  • 业务规则

  1. 显示当前我收到的所有预约请求，默认10条/页。
  2. 每条预约的取消时间取决于后台设置，后台设置默认为5天。
  3. 接受预约需要二次确认，确认文案：在线互动阶段接受的预约请求无法取消，您确定要接受吗？
  4. 拒绝接受预约需要二次确认，确认文案：您确定要拒绝该预约请求吗？
2.2.1.17 我的预约：发出的预约请求
  • 简要说明

查看当前用户发出去的预约请求，可以查看对方信息，发送消息以及撤回请求

  • 业务规则

  1. 撤回：取消用户发出去的预约请求，需要二次确认，确认文案：您确定要撤回该预约请求吗？
2.2.1.18 我的预约：我拒绝的预约请求/被拒绝的预约请求
  • 简要说明

查看被用户拒绝的预约请求，可以查看对方信息，发送消息，重新发送预约和删除操作

  • 业务规则

  1. 重新发送：再次对该预约对象发送预约请求
  2. 删除：删除该记录，无特殊业务逻辑
2.2.1.19 预约：选择了我的买家/展商
  • 简要说明

点击“选择了我的买家/展商”可查看在PSA阶段选择过我的，但是未能匹配成功的买家/展商列表，可以通过这个列表对之前选择过我的预约对象进行快速预约，通过这个列表进行预约不需要经过对方同意即可预约成功

  • 业务规则

  1. 点击“选择了我的买家/展商”可查看在PSA阶段选择过我的，但是未能匹配成功的买家/展商列表
  2. 通过该列表对之前选择过我的预约对象进行预约不需要经过对方同意即可预约成功
  3. 选择一个预约对象，点击预约进入选择预约时间段页面
  4. 仅双方都空闲的时间可以选择
2.2.1.20 预约：所有买家
  • 简要说明

点击“所有买家”，可以使用公司、联系人关键字和国家进行搜索预约对象，对与预约对象进行预约

  • 业务规则

  1. 点击所有买家，则进入所有买家的列表，可以使用公司、联系人关键字和国家进行搜索预约对象
  2. 预约操作与“选择了我的买家/展商”一样
